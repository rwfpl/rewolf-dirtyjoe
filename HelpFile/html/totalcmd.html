Total Commander plugin have the same functionality as stand-alone dirtyJOE (except "Register shell extension" feature, because it doesn't make any sense to put it into plugin). There is also cosmetic change in GUI, instead of main menu, there is "Plugin Options" button on the top of the window. With this button you can access some of the features:

  • Save File
  • Restore Debug Info
  • Settings
  • Help Topics
  • Opcodes Help
  • Check for Updates
  • Donate
  • About
