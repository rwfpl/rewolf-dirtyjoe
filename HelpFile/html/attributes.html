Editing Attributes

You can edit attributes in two ways:

  • by double clicking on specific attribute in a listbox
  • by selecting Edit Attribute option from context menu

Using context menu you can also:

  • Add a new attribute
  • Delete an existing attribute
  • View a text representation of attribute

Currently supported attributes:

  • InnerClasses (add/edit/view)
  • LocalVariableTable (add/edit/view)
  • ConstantValue (add/edit/view)
  • Signature (add/edit/view)
  • SourceFile (add/edit/view)
  • EnclosingMethod (add/edit/view)
  • Synthetic (add, empty attribute)
  • Deprecated (add, empty attribute)
  • Code (add/edit/view)
  • Exceptions (add/edit/view)
