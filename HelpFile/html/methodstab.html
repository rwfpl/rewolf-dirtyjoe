On Methods tab you can:

  • Edit and view all parameters of each method
  • Add/Edit/View/Delete attributes for each method
  • Add new method
  • Delete method
  • Copy disassembly to clipboard (right click on disassembly listbox)
Double click on the methods list will trigger Edit action for the first attribute from the Attributes list (e.g. Code, Exceptions). The same effect will be achieved by double clicking on the preview (bottom) panel.
