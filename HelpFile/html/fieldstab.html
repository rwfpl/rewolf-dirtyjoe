On Fields tab you can:

  • Edit and view all parameters of each field
  • Add/Edit/View/Delete attributes for each field
  • Add new field
  • Delete field
Double click on the fields list will trigger Edit action for the first attribute from the Attributes list (e.g. Signature, ConstantValue). The same effect will be achieved by double clicking on the preview (bottom) panel.
